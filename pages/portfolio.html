プロジェクト

トレードノートアプリ

トレードノート

投資家向けのトレード記録・分析アプリケーション。取引履歴の記録、パフォーマンス分析、チャート表示機能を提供します。

React Native Expo Node.js PostgreSQL
主な機能
  • 取引記録の作成・編集・削除
  • リアルタイムの価格データ表示
  • パフォーマンス分析とレポート生成
  • チャート表示とテクニカル分析
  • ポートフォリオ管理
  • プッシュ通知機能

使用技術

フロントエンド

React Native Expo JavaScript TypeScript

バックエンド

Node.js Express.js RESTful API

データベース

PostgreSQL SQL

開発・デプロイ
